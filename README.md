# Sound Platform

This is the monorepo for the Sound Platform, utilizing Firebase and multiple sub-apps (web, mobile, admin).

## Structure
- `/apps` - Sub-applications (web, mobile, admin)
- `/functions` - Firebase Cloud Functions
- `/packages` - Shared code, configurations, UI components, and validation
- `/firebase` - Firebase emulator configuration, rules, indexes, and seed data
- `/docs` - Project documentation
- `/scripts` - Development and build scripts
- `/tools` - Automation and helper tools
